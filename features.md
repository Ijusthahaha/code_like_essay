Here's a comprehensive summary of the syntax for your "Code Like Essay" project, based on your desired features:

### Variable Declaration and Assignment

- **Declare or Set a Variable**  
  `"set [variable] to [value]"` or `"define [variable] as [value]"`  
  Example: `set x to 10`

- **Supported Types for Values**
    - **Numbers**: `set x to number 10`
    - **Text**: `set x to string "hello"`
    - **Boolean**: `set x to true`

### Basic Operations

- **Addition/Subtraction**  
  `"display [x] add [y]"` or `"display [x] subtract [y]"`  
  Example: `display 5 add 10`

- **Multiplication/Division**  
  `"display [x] multiply [y]"` or `"display [x] divide [y]"`  
  Example: `display 10 multiply 2`

- **Increase/Decrease a Variable**  
  `"increase [variable] by [value]"` or `"decrease [variable] by [value]"`  
  Example: `increase x by 1`

### Conditional Statements

- **Conditional Start**  
  `"suppose that [condition], then [action]"`  
  Example: `suppose that x equals 10, then display "Equal"`

- **Alternative**  
  `"otherwise, [alternative action]"`  
  Example: `otherwise, display "Not equal"`

- **Comparison Operators**  
  `"greater than"`, `"less than"`, `"equals"`, `"not equals"`  
  Example: `suppose that x greater than 5`

### Loop Structures

- **Repeat a Block (Loop)**  
  `"repeat [action] until [condition]"`  
  Example: `repeat display x until x equals 0`

- **Loop with Increment/Decrement**  
  `"repeat [action] increase [variable] by [value] until [condition]"`  
  Example: `repeat display i increase i by 1 until i equals 10`

### Data Structures

#### Lists

- **Define a List**  
  `"set [list name] to list containing [elements]"`  
  Example: `set myList to list containing 1, 2, 3`

- **Add to List**  
  `"add [element] to [list name]"`  
  Example: `add 4 to myList`

- **Remove from List**  
  `"remove [element] from [list name]"`  
  Example: `remove 2 from myList`

#### Maps

- **Define a Map**  
  `"set [map name] to map with keys [key-value pairs]"`  
  Example: `set myMap to map with keys "name" equals "Alice", "age" equals 30`

- **Add Key-Value Pair**  
  `"add [key] equals [value] to [map name]"`  
  Example: `add "location" equals "City" to myMap`

- **Remove Key**  
  `"remove key [key] from [map name]"`  
  Example: `remove key "age" from myMap`

### Display Output

- **Print a Value**  
  `"display [value]"`  
  Example: `display myList`

This syntax is structured to be highly readable, following your project's goal of creating a programming language that feels like writing natural sentences.
